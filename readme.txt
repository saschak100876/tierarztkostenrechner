GOT-basierter Tierarztkostenrechner als Custom WordPress Plugin mit Shortcode, REST API, CSV-Import und eigenen Datenbanktabellen.

== Description ==

Der Shortcode [tierarztkostenrechner] rendert einen unabhängigen Rechner. Elementor ist optional und enthält keine Domainlogik.

== Installation ==

1. Plugin hochladen und aktivieren.
2. CSV-Daten im Adminbereich importieren.
3. Shortcode [tierarztkostenrechner] auf einer Seite einfügen.
